Ce repository est destiné au projet personnel "Modélisation des risques d'un portefeuille".
Le projet est mené conjointement avec Hugo PHOU.
Le projet comporte un fichier où est écrit le code, et un fichier où est écrit le "cours" (définitions, interprétations et calculs des notions abordées).

Sa première version est un brouillon ayant pour objectif de retranscrire les notions apprises, et programmer du code pour appliquer ces notions.
Sa version finale aura pour objectif un code propre et un guide pour l'utiliser.

Pour les créateurs du projet, l'objectif du projet est double :
- Apprendre, comprendre et se servir de certains outils statistiques et financiers utilisées dans la modélisation des risques d'un portefeuille;
- Apprendre à utiliser certaines bibliothèques et librairies, telles que scipy, matplotlib, pandas, ...

Pour l'utilisateur, les résultats du programme permettent de lui apporter des informations sur les risques qu'il porte. 
Ces résultats peuvent être plus ou moins pertinents, MAIS N'INCITENT EN AUCUN CAS À PRENDRE UNE DÉCISION PLUTÔT QU'UNE AUTRE.

LES RÉSULTATS DU PROGRAMME SONT PRÉSENTÉES À TITRE INFORMATIVE, ET NON DÉCISIONNELLE.
